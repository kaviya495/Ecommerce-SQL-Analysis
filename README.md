#E-Commerce Sales Analysis using SQL & Python

Project Overview:
This project analyzes e-commerce transactional data using SQL and Python to generate business insights on customer behavior and revenue trends.

Tools & Technologies:
- Python
- Pandas
- SQL (SQLite)
- Jupyter Notebook
- Matplotlib

Project Workflow:
1. Data loading and cleaning using Python
2. Created SQLite database from CSV datasets
3. Performed SQL queries for business analysis
4. Generated visualizations for sales trends

Key Analysis:
- Total Orders Analysis
- Revenue Analysis
- Top Customer Identification
- Monthly Sales Trend
- Payment Method Analysis

Business Insights:
- Identified peak sales months
- Analyzed customer purchasing behavior
- Evaluated revenue performance
